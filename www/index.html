// a walls.io experiment

Your social wall.
Eighty columns
of glory.

Walls.TUI streams your walls.io wall over a raw socket and renders it where you actually live: the terminal. Photos in 64 shades of half-block phosphor. Videos, frame-extracted and looping. Maps, rasterized tile by tile. Nine layouts. Zero browsers.

01 stdout, but make it social

      

        

          
▀▄▀

          
Real pictures. Real characters.

          
Every photo is rasterized into half-block pixels — the ▀ glyph, foreground up, background down — quantized to 64 shades and contrast-stretched. Chunky. Legible. Beautiful in a way JPEG forgot.

        

        

          
▶ ▶ ▶

          
Video, the honest way.

          
ffmpeg pulls ten frames straight off the source CDN, spread across the clip, cached on disk. They loop as a slideshow at 0.8 s per frame. It's not 60 fps. It's better: it's in your terminal.

        

        

          
◉

          
Cartography included.

          
Geotagged posts render on live OpenStreetMap tiles — fetched, composited, rasterized, and stamped with a glowing anti-aliased marker. A world map. Made of text. Your GIS friends will weep.

        

        

          
⇥ ←→ ↵

          
Keyboard-native.

          
Tab through posts, arrow through every hyperlink, Enter to open in the browser (fine, one browser use). OSC-8 escapes make links Cmd-clickable in modern terminals.

        

        

          
●━━

          
Live. Actually live.

          
A Socket.IO feed pushes new posts, edits, pins and moderation the moment they happen — the same protocol the real wall uses. No polling. No refresh. Just tail -f for your community.

        

        

          
$ _

          
One file. Anywhere.

          
npm run build compiles everything — TypeScript, React, the native renderer — into a single self-contained executable. Copy it to a kiosk, a Pi, a server rack. It just runs.

03 RTFM (it's a good M)

      

        
WALLS-TUI(1)User CommandsWALLS-TUI(1)

        

          
NAME

          
walls-tui — render a walls.io social wall in glorious text

          
SYNOPSIS

          
walls-tui [--wall id] [--network name] [--layout fluid|kiosk|map|world|ticker|theater|dashboard|channels|screensaver]

          
DESCRIPTION

          
Connects to the walls.io broadcaster, keeps a deduplicated, weight-sorted post buffer, and renders it live. Hidden posts vanish. Pinned posts float. Nobody touches a mouse.

          
KEYBINDINGS

          

            l cycle layout
            Tab focus post
            ←→ walk links
            ↵ open in browser
            Space next slide
            jk scroll
            r reload
            q quit
          

          
FILES

          
~/.cache/walls-tui/frames/ — extracted video frames
~/.cache/walls-tui/tiles/ — OpenStreetMap tiles

          
SEE ALSO

          
walls.io, opentui(7), your terminal's truecolor settings, the good old days

        

      

    


    
    

      
04 install in four lines

      

        

          

            
~ / anywhere-you-like
              
            

            
$ git clone https://github.com/michaelkamleitner/Walls.io-TUI.git
$ cd Walls.io-TUI
$ npm install
$ npm start

          

          

            
your wall, your rules
              
            

            
$ npm start -- --wall 139355 --layout kiosk
# any public walls.io wall id · any of the 9 layouts

          

          

            
or: one binary to rule them all
              
            

            
$ npm run build          # → ./walls-tui (self-contained, ~71 MB)
$ cp walls-tui /usr/local/bin/
$ walls-tui              # from anywhere. even over ssh.

          

        

        

          
// requirements

          
    
            
  • Node ≥ 20 + npm. Bun installs locally as a dev dep — nothing global.
    • 
            
  • A truecolor terminal. Ghostty, iTerm2, WezTerm, Kitty, Windows Terminal. 16 colors will not do 64 grays.
    • 
            
  • ffmpeg (optional) — unlocks video slideshows.
    • 
            
  • A walls.io wall. Yours, or someone's public one.
